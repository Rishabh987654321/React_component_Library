## Usage

<Source code={`import { Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter, Button } from '@rishabh_7775/react-component-library';

export function CardExample() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Card Title</CardTitle>
        <CardDescription>Card description text goes here.</CardDescription>
      </CardHeader>
      <CardContent>
        <p>Main content area</p>
      </CardContent>
      <CardFooter>
        <Button>Action</Button>
      </CardFooter>
    </Card>
  );
}`} language="tsx" />

## Accessibility

`Card` uses semantic layout containers (`CardHeader`, `CardTitle`, `CardDescription`, `CardContent`, `CardFooter`) with heading hierarchy (`h3`) and contrast-compliant borders.
