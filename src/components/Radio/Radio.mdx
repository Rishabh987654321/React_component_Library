## Usage

<Source code={`import { Radio, RadioGroup } from '@rishabh/ui';

export function RadioExample() {

      <Radio value="starter" label="Starter" />

## Accessibility

`RadioGroup` wraps radio controls inside a semantic `<fieldset>` with a `<legend>` header. Individual radio buttons use native `<input type="radio">` elements, enabling standard arrow-key navigation within the radio group for screen readers and keyboard users.
