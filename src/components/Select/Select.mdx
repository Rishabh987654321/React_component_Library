## Usage

<Source code={`import { Select } from '@rishabh_7775/react-component-library';

export function SelectExample() {
  return (
    <Select label="Country" defaultValue="us">
      <option value="us">United States</option>
      <option value="ca">Canada</option>
      <option value="uk">United Kingdom</option>
    </Select>
  );
}`} language="tsx" />

## Accessibility

Implemented using a styled native `<select>` element to preserve built-in platform accessibility, type-ahead keyboard navigation, and mobile dropdown overlays without introducing custom combobox bugs.
