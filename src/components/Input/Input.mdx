## Accessibility

The `Input` component generates an explicit `id` and `htmlFor` connection between the `<label>` and native `<input>` element. When an `error` message is provided, `aria-invalid="true"` is set on the input and `aria-describedby` links directly to the error message text `id` for screen reader announcement.
