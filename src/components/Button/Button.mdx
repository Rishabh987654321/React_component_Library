## Accessibility

Buttons are natively focusable and keyboard-operable (`Enter` and `Space` keys activate). Focus is visually indicated with a 2px accent-colored ring with a 2px offset (`focus-ring`). Loading states dynamically set `aria-busy="true"` on the native `<button>` element so screen readers announce the pending state, while the decorative spinner icon has `aria-hidden="true"`.
