## Accessibility

`Tabs` implements the WAI-ARIA Tabs pattern using `role="tablist"`, `role="tab"` with `aria-selected`, and `role="tabpanel"` linked via `aria-controls` and `aria-labelledby`. Employs a roving `tabIndex` (`tabIndex={0}` for active tab, `-1` for inactive tabs) with `ArrowLeft`, `ArrowRight`, `Home`, and `End` keyboard navigation.
