  const [open, setOpen] = useState(false);

  return (

## Accessibility

`Modal` uses `createPortal` to render directly to `document.body`, avoiding stacking context bugs. Integrates `focus-trap-react` for focus trapping inside open dialogs (`role="dialog"`, `aria-modal="true"`, `aria-labelledby`). Pressing `Escape` or clicking the backdrop closes the modal, restoring keyboard focus to the triggering element upon closure.
