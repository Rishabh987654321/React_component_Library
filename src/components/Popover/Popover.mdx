# Popover

Popover displays floating interactive panels relative to a trigger element.

## Usage

<Source code={`import { Popover, Button, Input } from '@rishabh_7775/react-component-library';

export function PopoverExample() {
  return (
    <Popover content={<div>Interactive Panel Content</div>}>
      {({ toggle }) => <Button onClick={toggle}>Toggle Popover</Button>}
    </Popover>
  );
}`} language="tsx" />

## Accessibility

`Popover` supports full keyboard interaction, closing on `Escape` key press or clicking outside the container.
